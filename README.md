# ResumeAnalyzer

A .NET API for analyzing resumes and matching them to job descriptions using AI.

## Features

- User authentication (register/login)
- Resume upload and parsing
- Job creation and matching
- AI-powered resume analysis

## Setup

1. Clone the repository.
2. Configure your database and API keys in `appsettings.json`.
3. Run the API with:

```bash
dotnet run
